Detailed Description

The Controller acts as an intermediary between the Model and the View. It handles user input and updates the Model accordingly and updates the View to reflect changes in the Model. It contains application logic, such as input validation and data transformation.

https://java-design-patterns.com/patterns/model-view-controller/
